Lisez-moi mise à jour version 3.3
**********************************

Ce patch n'est à installer que si vous avez une version 3.2

Pour une installation complète, le package d'installation Avatar.7z contient déjà ce patch.

Procédure d'installation :
	- Copiez le répertoire resources dans <SERVEUR>/
	- Au besoin, validez les remplacements fichiers
	- N'oubliez pas de changer la version dans le fichier de propriétés avatar.prop.
